Account Deletion / Hesap Silme - Ogi Lingo

You can delete your Ogi Lingo account directly inside the app.

Ogi Lingo hesabınızı uygulama içinden silebilirsiniz.

How to delete your account / Hesap nasıl silinir

  1. Open Ogi Lingo / Ogi Lingo'yu açın
  2. Go to Settings / Ayarlar bölümüne girin
  3. Tap "Hesabımı Sil"
  4. Confirm the deletion request / İşlemi onaylayın

Data deleted / Silinen veriler

  • User profile / Kullanıcı profili
  • Username / Kullanıcı adı
  • Friends list / Arkadaş listesi
  • Friend requests / Arkadaşlık istekleri
  • 1v1 match history / 1v1 maç geçmişi
  • 1v1 statistics / 1v1 istatistikleri
  • Leaderboard data / Sıralama verileri
  • Gameplay progress / Oyun ilerlemesi

Some anonymous technical or security-related logs may be retained temporarily for fraud prevention, debugging, legal compliance, or security purposes.
